## 8. 접근성

| 체크 | 기준 |
|---|---|
| 컬러 대비 | 본문 4.5:1, UI 3:1 |
| 포커스 표시 | outline 2px solid var(--brand-accent), offset 2px |
| 키보드 탭 순서 | 시각 순서와 논리 순서 일치 |
| 폼 레이블 | 모든 input에 `<label>` 연결 또는 `aria-label` |
| 에러 메시지 | `aria-describedby` 연결, `role="alert"` |
| 이미지 | 의미있는 이미지 `alt`, 장식은 `alt=""` |
| 아이콘 | 장식 `aria-hidden="true"`, 기능 `aria-label` |
| 모달 | `role="dialog"`, focus trap, ESC 닫기 |
| 스킵 링크 | "본문 바로가기" 제공 |
| Lang 속성 | `<html lang="ko">` |
