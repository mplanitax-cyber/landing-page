## 3. 잔여 리스크·주의 사항

1. **Formspree 폼 ID** — `action="https://formspree.io/f/YOUR_ID"` 교체 필수 (배포 전)
2. **협력사 로고 이미지** — 현재 SVG 텍스트 placeholder. 제휴계약 확보된 보험사만 실제 로고 교체
3. **사업자 정보 placeholder** — Footer 의 사업자등록번호·주소·전화번호 실제 값으로 교체 필요
4. **14세 미만 처리 금지 문구** — 개인정보처리방침 페이지(legal/privacy) 에서 문구 확정 필요 (v1.1)
5. **Lighthouse 성능 점수 미측정** — 배포 후 실측 요망
6. **공통 페이지 3종 (terms·privacy·board) 미구현** — v1.1 범위
7. **백엔드·관리자·FAQ TOP5 동적 집계** — v2 확정 이월
8. **준법 심의필 번호** — 클라이언트 테마 교체 시 Footer JSON 필드로 주입
